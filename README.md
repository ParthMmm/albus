# Impress

Find your new favorite album, and keep track of your albums.

## Features
- User registration
- Album search
- Log albums
- Profiles
- Dark mode compatibility
- Responsive design


## Built With
- React
- Next.js
- Chakra UI
- MongoDB
- Express
- Node.js
